# 🎤 IA Entrevistador: Seu Simulador Inteligente de Entrevistas

Projeto desenvolvido como parte do **Desafio de Projeto da DIO - Inteligência Artificial na Prática**.  
O objetivo é criar um **simulador inteligente de entrevistas técnicas**, capaz de gerar perguntas relevantes, avaliar respostas e ajudar o usuário a se preparar para processos seletivos em tecnologia.

---

## 🚀 Funcionalidades
- Simulação de entrevistas técnicas com IA
- Geração de perguntas personalizadas por área (backend, frontend, dados, IA)
- Avaliação automática das respostas
- Feedback prático e direcionado para evolução

---

## 🛠️ Tecnologias Utilizadas
- Python 3.10+
- Bibliotecas: `transformers`, `nltk`, `streamlit`
- Engenharia de Prompt para IA Generativa
- GitHub Actions para automação

---

## 📂 Estrutura
- `src/` → Código principal da IA entrevistadora
- `data/` → Banco de perguntas e exemplos
- `docs/` → Documentação adicional
- `requirements.txt` → Dependências do projeto
