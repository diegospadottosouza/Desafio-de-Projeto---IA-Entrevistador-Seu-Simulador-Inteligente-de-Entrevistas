# Desafio-de-Projeto---IA-Entrevistador-Seu-Simulador-Inteligente-de-Entrevistas
   IA Entrevistador é um simulador inteligente de entrevistas que utiliza IA generativa e engenharia de prompt para criar perguntas técnicas, avaliar respostas e fornecer feedback personalizado, ajudando usuários a se preparar melhor para processos seletivos em tecnologia.
